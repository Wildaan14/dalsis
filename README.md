# Project DALSIS-AI

Digital Analytics for Livestock Sustainability and Intelligence System.

## Architecture
- **Frontend**: React + Vite + TailwindCSS
- **Backend**: Node.js + Express
- **AI/ML**: Python (Random Forest) + Google Earth Engine API
- **Standard**: IPCC Tier 1 (2019) & IPCC AR6

## Setup
1. Clone the repository.
2. Install frontend dependencies: `cd frontend && npm install`
3. Install backend dependencies: `cd backend && npm install`
4. Set up environment variables in `backend/.env` and `frontend/.env`.
